# هدف الوركشوب

في نهاية الوركشوب هتكون قادر:

* تكتب **Embedded C**
* تفهم **Microcontrollers**
* تعمل Drivers زي:

  * GPIO
  * LCD
  * Keypad
  * Timers
  * ADC
  * PWM
* تستخدم بروتوكولات:

  * UART
  * I2C
  * SPI
* وتعمل **Embedded Project كامل**
